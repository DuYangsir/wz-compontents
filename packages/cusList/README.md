# wz-cus-list

### 使用说明:

```html
  <wz-cus-list>
    <wz-cus-list-item>
      <!-- your code -->
    </wz-cus-list-item>
  </wz-cus-list>
```

### 配置参数:

| 参数  | 值                                                        |
| ----- | :-------------------------------------------------------- |
| color | default: 'blue'; 其他值('red', 'green', 'yellow', 'gray') |
