# wz-cus-list

### 使用说明:

```html
  <wz-cus-list>
    <wz-cus-list-item>
      <!-- your code -->
    </wz-cus-list-item>
  </wz-cus-list>
```

<br />

### 配置参数:

| 参数  | 类型   | 值                                                       |
| ----- | :----- | :------------------------------------------------------- |
| color | String | 默认值: 'blue'; 其他值('red', 'green', 'yellow', 'gray') |
